# Push2heroku

Push to heroku.

## Installation

Add this line to your application's Gemfile:

    gem 'push2heroku'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install push2heroku

## Usage

First copy `push2heroku.yml` from [here]().

`rake push2heroku`

push2heroku uses `cedar` stack.
